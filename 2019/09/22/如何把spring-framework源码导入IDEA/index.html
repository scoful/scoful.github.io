《如何把spring-framework源码导入IDEA》

  1. spring-framework用的是Gradle编译的，所以本地要先安装Gradle，具体方式请参照其他文章。

  2. 需要安装Git和JDK1.8，这里有个必须注意事项，JDK的版本必须要大于60，命令行 java -version，就能看到java version ，例如：java version “1.8.0_221”，低于60的必须升级才能成功。顺便提一下，绿色版本JDK怎么提前看版本，文件夹目录下，有个叫release的文件，打开就能看到，这种适用于安装配置JDK前，不知道JDK的版本信息。

  3. 环境变量JAVA_HOME需要配置。

  4. 从官方仓库 https://github.com/spring-projects/spring-framework Fork 出属于自己的仓库。

  5. 在本地找个目录放置spring-framework源码，git命令下载到本地：git clone xxx(你自己fork下后生成的地址)。

  6. 下载好后，找到文件build.gradle，打开，查找关键词：repositories，加入以下2行命令，用华为节点加快下载速度

    mavenLocal()
maven { url "https://mirrors.huaweicloud.com/repository/maven/"}

    命令：cd spring-framework，编译命令：gradlew build，需要很长时间。

  1. build成功后，打开IDEA，Import Project，over。
